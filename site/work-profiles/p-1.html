Infinite Retries

Infinite Retries is a video game based web app, currently under development. I co found this idea with Tristan Quintana, I am in charge of design and development while Tristan works marketing and business. This side project is a fun but demanding goal of ours. It is teaching us what it takes to build and launch a scalable application.

We are launching our first landing page this fall, hoping to follow up with a basic React prototype. The prototype is to test user experience, proof of concept, and gain some preliminary exposure. If that goes well we are looking to develop Infinite Retries with a MERN stack (Mongo, Express, React, Node). I am really excited and motivated to see where this experience takes my abilities.

Project Details:

  • Bootstrap
  • Sass
  • MERN ( Mongo, Ember, React, Node)
  • Web application
